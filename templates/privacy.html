Confidentialité

Confidentialité

Montreal MCM est un petit site de découverte de mobilier vintage. La collecte de données reste volontairement limitée: assez pour faire fonctionner les favoris, les recherches sauvegardées, les rafraîchissements et les statistiques de base, sans profils publicitaires.

Favoris et recherches sauvegardées

Les favoris anonymes, boutiques sauvegardées et recherches sauvegardées utilisent un cookie de première partie signé. La base de données garde seulement une clé propriétaire hachée avec les identifiants des éléments sauvegardés. Il n'y a pas de compte utilisateur public pour cette fonctionnalité.

Statistiques

Montreal MCM conserve des statistiques agrégées de pages vues par jour, type de page, chemin et langue. Le site ne stocke pas les adresses IP brutes, les agents utilisateurs ou les référents dans cette table applicative. Cloudflare traite aussi les requêtes et journaux nécessaires pour servir, sécuriser et mesurer le site.

Témoins

Le site utilise des témoins de première partie pour la langue, la session Flask et l'identité anonyme durable des favoris/recherches. Aucun script publicitaire tiers n'est chargé par Montreal MCM.

Conservation

Les données sauvegardées anonymes et les statistiques agrégées sont conservées tant qu'elles sont utiles au fonctionnement et à l'amélioration du site. Les données de catalogue proviennent des sources publiques des boutiques et gardent leur provenance.

Privacy

Privacy

Montreal MCM is a small vintage furniture discovery site. Data collection is deliberately limited: enough to run favourites, saved searches, refresh operations, and basic site analytics, without advertising profiles.

Anonymous favourites and saved searches

Anonymous favourites, saved shops, and saved searches use a signed first-party browser cookie. The database stores only a hashed owner key with the IDs of saved items. There are no public user accounts for this feature.

Analytics

Montreal MCM stores aggregate page-view counts by day, page type, path, and language. The app does not store raw IP addresses, user agents, or referrers in this analytics table. Cloudflare also processes requests and logs needed to serve, secure, and measure the site.

Cookies

The site uses first-party cookies for language, the Flask session, and durable anonymous favourite/search identity. Montreal MCM does not load third-party advertising scripts.

Retention

Anonymous saved data and aggregate analytics are retained while they remain useful for operating and improving the site. Catalogue data comes from public shop sources and keeps source provenance.
